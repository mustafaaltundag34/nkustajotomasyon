NKU Staj Otomasyon Sistemi Icin yapilmasi gerekenler 
------------------------------------------------------
1- Sisteme XAMPP Uygulamasi kurulur 

	Asıl Link	: https://www.apachefriends.org/xampp-files/7.3.27/xampp-windows-x64-7.3.27-0-VC15-installer.exe

2- XAMPP Uygulamasi calistirilir ve Apache ile MYSQL Portlari Start Edilir 

3- Phpmyadmin üzerinden Stajdb adinda bir veritabani olusturulur ve Stajdb.sql dosyasi import edilir 

	Asıl Link	: https://drive.google.com/file/d/1HRyjN2ykLCn4y_SwmFPH6QOv8_YsvyVt/view?usp=sharing


4- Repository üzerindeki Nkustajotomasyon klasoru c:/XAMPP/Htdocs/ klasoru altına kopyalanir 

	Asıl Link 	: https://drive.google.com/file/d/1BqgvgECMrcKF5DVG4A6PLd3Tk7TSYncb/view?usp=sharing
	Alternatif Link	: https://github.com/mustafaaltundag34/nkustajotomasyon
